Aplicacion desarrollada en kotlin utilizando MVVM, clean code, Unitest, fragments, inyeccion de dependencias, stateFlow y corrutinas con 3 pantallas utilizando navigation component y mappers:
1. Horoscopo: Utilizando un recycler view aparecen todos los signos zodiacales, al seleccionar uno, despues de una animacion, aparece la prediccion del dia actual.
2. Luck: Una ruleta que al deslizar de derecha izquierda o viceversa hara una animacion y al terminar lanzara otra animacion con la tarjeta, despues, mostrara
   una prediccion aleatoria (Utilizando Retrofit e interceptors) que puedes compartir con su respectiva imagen en cualquier aplicacion.
3. Palmistry: Utilizando la camara con sus respectivos permisos (Camera X) muestra una imagen de una mano para que sea enpalmada con la tuya
